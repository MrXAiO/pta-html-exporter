# PTA HTML Exporter

一个用于 PTA / Pintia 平台的浏览器用户脚本，可以将当前题目页面导出为**可编辑、可跳转、可打印、可离线保存的 HTML 文件**。

相比传统的 `.txt` / `.doc` 导出方式，本项目更注重保留原页面结构、题目样式、选项状态、代码块、图片链接和左侧题号导航，适合个人复习、错题整理、资料备份和离线查看。

> 本项目仅用于学习备份、错题整理和离线复习。  
> 导出的 HTML 文件中的编辑和答案修改只作用于本地文件，不会提交到 PTA，也不会影响官方成绩。

---

## 功能特性

- 导出 PTA 当前页面中的真实题目块
- 保留题目原始 HTML 结构
- 保留选项、答案状态、代码块、图片和链接
- 自动生成题号导航栏
- 支持题号快速跳转
- 支持本地编辑题目内容
- 支持重新选择答案并同步答案块
- 支持隐藏 / 显示答案
- 支持打印导出的 HTML
- 支持保存修改后的 HTML
- 支持手机和平板自适应显示
- 无后端、无依赖、无需登录额外服务
- 所有导出操作均在浏览器本地完成

---

## 项目特点

### 保留图片和页面结构

很多导出工具只支持导出纯文本或简单文档格式，遇到图片、代码块、复杂排版时容易丢失内容。

本项目采用 HTML 原样导出的方式，尽量保留 PTA 页面中的：

- 题目正文
- 选项区域
- 代码块
- 图片资源
- 超链接
- 页面样式
- 答案状态
- 评测信息

因此更适合包含图片、表格、代码和复杂排版的题目。

## 安装方式

### 方式一：通过 Tampermonkey 安装

1. 安装浏览器扩展 Tampermonkey
2. 新建用户脚本
3. 删除默认内容
4. 粘贴本项目中的脚本代码
5. 保存脚本
6. 打开 PTA 对应题目页面
7. 页面右下角出现“导出”按钮即代表加载成功

## 支持页面

当前匹配规则：

```js
// @match https://pintia.cn/problem-sets/*/exam/problems/*
```

也就是说，本脚本主要适用于类似下面的 PTA 页面：

```text
https://pintia.cn/problem-sets/xxx/exam/problems/xxx
```

如果 PTA 页面结构发生变化，可能需要更新题目识别规则。

---

## 使用方法

1. 打开 PTA 题目页面
2. 等待题目内容加载完成
3. 点击页面右下角的“导出”按钮
4. 点击“导出可编辑 HTML”
5. 浏览器会自动下载一个 `.html` 文件
6. 使用浏览器打开该 HTML 文件即可离线查看

导出的 HTML 页面支持：

- 左侧 / 顶部题号导航
- 点击题号快速跳转
- 编辑题目内容
- 修改选项答案
- 显示或隐藏答案
- 打印页面
- 保存修改后的 HTML 文件

## 移动端适配

从 `14.7.0` 版本开始，导出的 HTML 页面增加了手机和平板适配。

### 桌面端

桌面端会保留经典布局：

```text
左侧题号导航 + 右侧题目内容
```

### 平板横屏 / 小屏桌面

平板横屏或较窄屏幕下：

- 左侧栏宽度会自动缩小
- 正文区域取消固定最小宽度
- 题目内容可自适应屏幕宽度

### 手机 / 平板竖屏

手机或平板竖屏下：

```text
顶部题号导航 + 下方题目内容
```

这样可以避免左侧栏占用过多宽度，提升小屏幕阅读体验。

## 隐私说明

本项目重视用户隐私。

脚本本身：

- 不上传题目内容
- 不上传答案内容
- 不读取账号密码
- 不请求外部 API
- 不向第三方服务器发送数据
- 不依赖后端服务
- 导出文件完全在浏览器本地生成

脚本权限：

```js
// @grant none
```

这意味着脚本没有申请额外的 Tampermonkey 特权 API。

## 关于图片保存

本项目会尽量保留题目中的图片资源。

目前的处理方式是：

- 将图片地址转换为绝对地址
- 保留图片在导出 HTML 中的显示能力
- 设置图片懒加载
- 设置图片引用策略

需要注意：

> 当前版本并不是将图片文件真正打包进 HTML。  
> 如果原图片地址失效、需要登录权限或网络不可访问，导出的 HTML 中图片可能无法显示。

未来计划支持：

- 图片转 Base64
- 单文件完整离线保存
- 断网环境下仍可查看图片

## 常见问题

### 为什么导出的 HTML 里图片有时无法显示？

因为当前版本保留的是图片链接，而不是把图片文件嵌入 HTML。

如果图片需要登录权限、原链接失效、网络不可访问，图片可能无法正常显示。

后续版本可能会增加图片 Base64 内嵌模式。

---

### 为什么不是导出为 Word 或 TXT？

TXT 会丢失结构、样式、图片和代码块格式。

Word 虽然适合编辑，但网页复杂结构转换成 Word 后经常出现排版错乱。

HTML 更接近 PTA 原页面，能更好保留：

- 原始结构
- 题目排版
- 图片
- 代码块
- 选项状态
- 链接
- 样式

因此本项目选择导出为 HTML。

---

### 修改导出的 HTML 会影响 PTA 成绩吗？

不会。

所有修改都只发生在导出的本地 HTML 文件中，不会提交到 PTA，也不会改变官方成绩。

---

### 为什么手机上左侧栏变成顶部栏？

因为手机屏幕较窄，如果继续使用桌面端左侧固定栏，会导致正文区域过窄、横向滚动严重。

因此移动端会自动切换为顶部题号导航。

---

### 支持哪些浏览器？

- Chrome + Tampermonkey
- Edge + Tampermonkey

- Firefox + Tampermonkey

- Chromium 系浏览器 + Tampermonkey
